Postup instalace

Aplikácácia vyžaduje Node.js, podporný nástroj npm pre kompiláciu frontendu, a Javu vo verzii 8 pre spustenie backendu

Inštalácia a spustenie:

  • Nainštalovanie knižníc potrebných pre frontend: 
                    cd client && npm install
  • Kompilácia kódu frontendu: 
                    npm run-script build
  • Kompilácia a spustenie servru: 
                    cd server
                ./gradlew bootRun

Pre zjednodušenie sú tieto kroy implementované v Makefile súbore, takže pre kompiláciu a spustenie projektu je nutný len príkaz: 

            make

Serverová aplikácia vykoná všetky potrebné kroky pre inicializáciu lokálnej sqlite databázy
